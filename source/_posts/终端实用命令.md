查看该目录下各个文件和子目录各占多少空间
```
du -sh *
```

批量复制文件
```
cp *.png *.jpeg *.gif /destpath (剪切的话 cp换成 mv)
```

快速创建便笺
选中文字，然后
```
shift + command + Y
```
